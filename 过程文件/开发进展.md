# 币安合约横盘监控开发进展

## 核心目标
监控全网 Binance USDT 永续合约中长期横盘、具备爆发潜力的标的。

## 进展日志
*   **初期调研**: 发现免费的前端工具（如 Coinglass）无法提供针对 Binance 的“横盘时间段”持续长短的榜单降序。
*   **网络攻坚 (Proxy Bypass)**: 编写了用于测试各个公共反代的脚本，确认 90% 节点被封。选用自建高级版 Cloudflare Worker 代理方案（带 `x-mbx-apikey` 保留及 Host 伪装，已由用户提供高质量代码并绑定 `binance.794988.xyz`）。
*   **引擎构建 (Sideways Scanner)**: 编写了原生 Python 计算引擎 `sideways_scanner.py`。
    *   **策略**: 时间从最新 K 线向后追溯计算，统计价格未越界 `2%` 振幅内的 K 线的累计连击数。
    *   **效能**: `< 5s` 内通过 API 瞬间并发抓取 300+ 虚拟货币的 `1h` 级别K线，并在内存完成排位。
*   **云架构**: 提供 Github Actions workflow `sideways_monitor.yml`，规避个人网络拦截，无头定时跑批。
